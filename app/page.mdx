# Introduction
Spend less time setting up libraries in your project and more time being productive.

Lib0 is the shadcn for libraries and packages. It provides a set of pre-built components that can be easily customized to fit your needs. Zero is currently under development and more libraries are being added. 

## Libraries

The libraries section provides one time run commands to include a specific library or package in your project.

## Recipes

The recipes section provides code snippets to combine multiple technologies and include them into your application.

# Better Auth
The most comprehensive authentication framework for TypeScript.

## Usage

Better Auth can be installed in your framework specific application. Use the appropriate installation command for your framework. All of the installation steps are from the official library documentaition. 

##### Next.js Pages Router
```sh copy
bun x --bun shadcn@latest add "https://lib0.vercel.app/r/better-auth-next-pages"
```

##### Next.js App Router
```sh copy
bun x --bun shadcn@latest add "https://lib0.vercel.app/r/better-auth-next-app"
```

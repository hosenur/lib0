# Twilio

Twilio is a cloud platform that provides APIs for adding messaging, voice, and video communication features to applications. It lets developers integrate SMS, calls, and other communication tools without building telecom infrastructure.

<Callout>If you are using any backend framework or Nextjs use the node command, if you are using any vite based meta framework like TanStack Start or React Router use the Vite implementation</Callout>
